About DARPAN

DARPAN (Service Management and Orchestration) is an intelligent, containerized network visualization and analytics testbed built for 5G architectures. It aggregates real-time metrics across Physical (O-RU), Radio Access (O-DU/CU), and Core (5GC) networks, providing full network visibility and closed-loop control.

By leveraging an in-memory EWMA baseline anomaly detector and least-squares regression forecasting, DARPAN predicts resource exhaustion and link degradation, allowing operators to proactively mitigate service disruptions before they impact subscribers.

Key Features

  • 🌐

    Dynamic Topology

    Interactive tree-graph rendering live connection edges, traffic flows, and dynamic cell outage indicators via Cytoscape.js.

  • 📉

    Unified KPI dashboarding

    High-resolution timeseries panels and gauges displaying throughput, RRC setup success, and core resource loads.

  • 💡

    Intelligent Forecasting

    Short-horizon forecasting models tracking drift metrics and generating automated alerts when values exceed 2.5 standard deviations.
